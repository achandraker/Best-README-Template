# Xe-Flake

<!-- ABOUT THE PROJECT -->
## About The Project

Quick & Easy migration of tables and database from RDBMS to snowflake.

## Description

Quick Data movement from any source database to snowflake just by specifying the table names through UI without any need to define schema or write code.

## Getting Started
How to run this in your local system.
Steps mentioned below are w.r.t windows machine as currently below setup has been tried in windows 11 os

### Dependencies/prerequisites 

* Python 3.7 or higher
* Mysql latest version
* Install python dependencies using requirements.txt
* AWS cloud account with S3, Lambda, Glue and Secret Manager access

### Steps

* Open MySQL create a database and tables which you want to upload in snowflake. As an option you can also use a web based online Mysql RDBMS
Create your credentials, login into it and create a database and table.
* Store your MySql login credentials inside AWS secret manager by creating a secret in your AWS (We are storing it here so that we can call it 
directly from our python code in place of including credential information in code).
